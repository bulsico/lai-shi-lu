# 加密市场背景参考 (2023–2026)

> 供 AI 分析 Hyperliquid 交易员时作背景参考。
> 此文档总结公开可知的市场规律、行为陷阱和技术信号，不包含任何个人仓位或账户数据。

---

## 市场阶段总览

### 2023–2024：MAG7集中牛市 + 加密反弹期

- 美股整体由MAG7（NVDA、MSFT、GOOGL、META、AMZN、AAPL、TSLA）驱动，其他499只标普成分股滞后约40个百分点
- BTC从$16K反弹至$70K+，叙事驱动：BTC ETF审批预期 → 2024年4月减半 → 机构进场
- 加密板块内部：BTC主导期，alt coin跟随BTC上涨但弹性更大，跌幅也更大
- 这一阶段做多BTC/ETH持有者普遍盈利，过度交易的反而不如持有

### 2024年4月：BTC减半

- 历史规律：减半后12-18个月通常是bull run顶点区域，随后进入熊市周期
- 历次减半后熊市跌幅：-70% 到 -85%（从ATH算起）
- 2024减半到2026年4月，当时跌幅约-37%，意味着如果历史规律成立，下行风险仍存

### 2025年下半年–2026年初：波动加剧期

- MAG7 vs 中型SaaS出现结构性分化：MAG7上涨15–40%，中型SaaS下跌30–75%
- AI叙事分化为赢家（超大规模云计算）和输家（被替代的SaaS工具）
- BTC 50/200日均死亡交叉（death cross）于2025年11月16日形成，持续5个月以上
- ETH、SOL、其他主流alt在death cross期间全部跑输

### 2026年3–4月：TACO循环

- "TACO"（Trump Triggered Tactical Organic Advance）：地缘政治紧张 → 缓和 → 加密+15–25%短线反弹
- 模式特征：每次TACO循环比上次更温和，说明资金在每次反弹时加仓更重
- 催化剂：伊朗危机、FOMC暂停加息、美国CLARITY稳定币法案、霍尔木兹海峡重开
- 重要提示：短线反弹不代表趋势反转，mid-term依然受BTC death cross压制

## Hyperliquid 平台特有规律

### 资金费率信号

| 资金费率状态 | 含义 |
|-------------|------|
| BTC年化 >10% | 零售追多，短期回调风险上升 |
| BTC年化 0–3% | 机构均衡态，无方向信号 |
| BTC年化 <0% 或极低 | 做多资金耗尽，盘整信号 |
| alt年化 <-8%（7日）且距ATH >50% | 空头拥挤，挤空行情可能来临 |
| alt年化 >50%+ | 做多极度拥挤，做多成本极高，可能是顶部信号 |

### OI/成交量比（流动性陷阱）

- OI/日成交量 > 5–8倍 = 流动性极差，任何真实规模的仓位都难以安全出入
- 常见陷阱：低流动性alt coin被某条alpha推文带动，散户争相追入，但实际成交深度不足以支撑
- 经验规则：日成交量 <$10–20M 的合约，谨慎开仓

### 零售散户在HL最常犯的错误

1. **在挤空行情前空弱alt** — 空头已大量堆积的标的，任何轻微利好就能触发踩踏
2. **忽略趋势状态直接入场** — 价格在SMA7之上但低于SMA50，不是熊市信号，不适合做空
3. **用加密对冲宏观风险** — 同时做空BTC表达宏观悲观，又做多某个alt表达基本面看好，两个逻辑互斥，执行时都乱了

## 关键技术信号：死亡交叉 / 金叉

### 什么时候死亡交叉真的很重要

- **BTC 50日/200日死亡交叉**：当BTC形成50/200死亡交叉时，整个加密市场都处于结构性下降趋势
  - 历史上，在BTC death cross期间做多alt的成功率显著低于平均水平
  - 信号：不要在BTC death cross期间新开alt多头，除非满足下面的挤空条件
  - 解除信号：BTC 50日均线重新站上200日均线（golden cross确认）

- **挤空例外条件**（三个条件同时满足才考虑）：
  1. 7日资金费率 ≤ -8%（空头严重拥挤）
  2. 30日资金费率为负值
  3. 距历史最高点跌幅 >50%

### 个股/alt级别的金叉/死亡交叉

- 单个alt的金叉信号，在BTC death cross大背景下效力大打折扣
- 需要"板块 + 个股"双重确认才可信：BTC先恢复结构性上涨，再看alt自身信号

## 交易员行为模式（普遍规律，非个人案例）

### 致命的行为陷阱

#### 1. 亏损后加仓（越亏越多）
- 生物本能：亏损时大脑会提高风险偏好试图"追回来"
- 结果：小亏变大亏，大亏变清仓
- 防守方式：提前设置机械止损，不给自己留"我再看看"的空间

#### 2. 活动性陷阱（越交易越亏）
- 规律：每天3笔精选交易（高胜率）的期望收益，远高于每天9笔随机交易（低胜率）
- 数字：3笔×70%执行质量 >> 9笔×50%执行质量（交易次数多了3倍，但收益只有三分之一）
- 结论：行情不明朗时，不交易也是一种选择

#### 3. 沉没成本锚定
- 表现：因为"已经亏了很多"而不愿止损，心里默默等解套
- 正确逻辑：当前该不该持有，和你的成本没有关系
- 检验：把仓位清零，如果现在还会买入，那就继续持有；如果不会买入，那就止损

#### 4. 论文陷阱（thesis trap）
- 表现：原始看好的逻辑已经被市场证伪（价格跌穿止损位、催化剂落空），但还在持有
- 逻辑：论文死了，仓位也应该结束
- 常见形式：某个事件驱动的交易，事件已经发生且不利，但还在"等等看"

#### 5. 过度分散 + 注意力稀释
- 同时跟踪10+个仓位，每个都"了解一点"，等于每个都不够了解
- 每年5–10个"大胆但有据可查的赌注"的期望收益，通常高于40个"分散风险的小仓位"

### 有效的防守规则

1. **所有仓位必须有止损**：价格止损或事件止损（催化剂落空即平仓），二选一
2. **死亡交叉期间不开新多**：BTC 50/200死亡交叉活跃时，不新开加密多头（除非满足挤空三条件）
3. **单次仓位不超过总资产的5%**：超过这个比例，一次大亏就会影响后续决策
4. **提前承诺资金部署节奏**：在事情发生前决定好"如果跌到XX，我买YY多"，而不是实时情绪决策
5. **区分趋势跟随和逆势抄底**：如果是趋势跟随，应该在趋势确认后（金叉、价格站上均线）入场；如果是逆势抄底，止损更重要

## 市场结构认知框架

### 哪种行情适合哪种策略

| 市场状态 | 适合策略 | 不适合策略 |
|---------|---------|----------|
| BTC牛市 + 金叉 | 持有BTC/ETH现货，跟随趋势做多alt | 频繁高抛低吸、做空 |
| BTC牛市 + alt轮动 | 阶段性做多轮动标的，及时止盈 | 持有过期叙事的alt，忽视轮动信号 |
| BTC横盘 + 资金费率中性 | 观望或超短线，控制仓位 | 重仓开多，用杠杆"赌突破方向" |
| BTC死亡交叉 + 熊市 | 轻仓、止损、保留现金 | 追弹、加仓、"这次不一样" |
| 地缘政治催化剂驱动的短线反弹 | 超短线参与，提前设好止盈止损 | 把短线行情当成趋势逆转信号长线持有 |

### 集中度风险的慢性毒药

- 50%仓位在单一资产，在高相关性牛市里感觉"很安全"（因为涨了）
- 相关性崩塌后，才意识到等于把所有鸡蛋放在一个篮子里
- 规律：**集中缓慢发酵，然后突然爆发** — 长达1-2年没有问题，然后一个月里问题全来了
